Temario HTML:

  1. Elementos
    • <h1> <h2> <h3>Títulos de varios tamaños
    • <p> Párrafos
    • <header> Encabezado de la web
    • <nav> agrupa enlaces a otras páginas
    • <body> abarca todo el contenido de la web
    • <aside> abarca el lateral de una web
  2. Atributos
    • <a href=""> enlace a otra página
    • <img src="">Para poner imágenes
    • width=""anchura de una imagen
    • height="" altura de una imagen
  3. Atributos de enlace
    • _selfabre en la misma pestaña
    • _blank abre en otra pestaña
    • _top abre el documento en el mismo body de la web
  4. Tablas
    • <table>crea una tabla
    • <th>define el encabezado de la tabla
    • <tr>define línea de la tabla
    • <td>define celda de la tabla
  5. Listas
    • <ul>lista desordenada
    • <ol>lista ordenada
    • <dt>lista descriptiva
Para más información sobre el temario mira esta guía rápida.

Temario CSS:

  1. Selectores
    • * { } selector universal
    • p { } selector de etiqueta
    • .clase{ } selector de clase
    • #id{ } selector identificador
    • .clase p{ } selector descendiente
    • .clase, p{ } Combinación de selectores
  2. Box model
    • { width: } manipula la anchura
    • {height: } manipula la altura
    • { border: } aplica estilo de borde de la caja
    • { margin:} aplica espacio entre los elementos
    • { background: } aplica un fondo a los elementos
  3. Modificar texto
    • { color: } cambia color de un texto
    • {text-align:} centra el texto
    • {vertical-align: } texto vertical
    • {font-family: } Tipo de letra
    • {font-size: } Tamaño de letra
    • {font-weight: } Peso (normal, negrita...)
    • {text- decoration: } Mostrar un texto en mayúsculas, minúsculas o la primera letra de cada palabra en mayúsculas.
Para más información sobre el temario mira esta guía rápida.
